# UX e acessibilidade

Este projeto apresenta uma landing page sobre UX e acessibilidade, com foco em interfaces inclusivas, contraste, semântica e navegação por teclado.

## Como visualizar

Abra o arquivo index.html em um navegador ou sirva a pasta com um servidor simples, como:

```bash
python3 -m http.server 8000
```

Depois acesse http://127.0.0.1:8000/.
